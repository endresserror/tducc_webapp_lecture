# Space Invaders Web App - Webアプリ開発講座

このプロジェクトはWebアプリ開発講座の一環として作成された、クラシックなスペースインベーダーゲームのWeb実装です。PythonとFlaskを使用して構築されており、ブラウザ上でゲームを楽しむことができます。

## プロジェクト概要

このプロジェクトは、コンピュータクラブ部のwebアプリ講座の初回に向けて作成したwebアプリとは何かを説明するために用意しました。

## 主要ファイルの説明

### static/js/game.js
このファイルがゲームの主要なロジックを含むJavaScriptファイルです。ここでは以下の変数を修正することでゲームの挙動を調整できます：

- `enemyMoveDelay`: 敵の移動速度（値が小さいほど速く動く）
- `speedFactor`: 敵の移動距離の倍率（値が大きいほど長く動く）
- `lives`: プレイヤーの初期残機数
- `player.health`: プレイヤーの体力値
- `player.speed`: プレイヤーの移動速度

### static/css/style.css
ゲームの見た目を制御するCSSファイルです。背景色やテキストのスタイルなどをカスタマイズできます。

### templates/game.html
ゲーム画面のHTMLテンプレートです。キャンバス要素の設定やJavaScriptファイルの読み込みなどが定義されています。

### app.py
Flaskアプリケーションのエントリーポイントです。ルーティング設定やサーバー側の処理が記述されています。

## セットアップ手順

1. **リポジトリをクローン**:
   ```
   git clone <repository-url>
   cd space-invaders-webapp
   ```

2. **依存ライブラリのインストール**:
   Python環境を準備し、以下のコマンドを実行:
   ```
   pip install -r requirements.txt
   ```

3. **アプリケーションの起動**:
   ```
   python app.py
   ```
   アプリケーションは `http://127.0.0.1:5000/` で起動します。

## ゲームの操作方法

- **移動**: 左右の矢印キー (← →) でプレイヤーの宇宙船を移動
- **発射**: スペースキーで弾を発射
- **一時停止**: Enterキーでゲームの一時停止/再開
- **目標**: 敵の攻撃を避けながらすべての敵を撃破する

## 講座での活用方法

この教材は以下のような学習目的で活用できます：

## 変数の変更例

例えば、以下のように変更することでゲームの難易度を下げることができます：

```javascript
// 敵の移動速度を遅くする（30から60に変更）
let enemyMoveDelay = 60;

// 敵の移動距離を短くする（1.0から0.7に変更）
let speedFactor = 0.7;

// プレイヤーの体力を増やす
this.health = 20; // デフォルトは10
```
